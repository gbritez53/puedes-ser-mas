# 🚀 PUEDES SER MÁS — Landing Page (Diplomado Coach Emprendedor)

Landing page de producción premium y alto impacto comercial, diseñada bajo los principios ontológicos de la transformación del **SER** y la superación humana. Este proyecto ha sido estructurado mediante desarrollo guiado por especificaciones (**SDD - Spec-Driven Development**) utilizando agentes autónomos (`gentle-ai`), garantizando una arquitectura ultraeficiente, libre de deuda técnica y con un rendimiento impecable (Score de 100 en Core Web Vitals).

---

## 🛠️ Stack Tecnológico

El proyecto exprime las tecnologías más modernas y veloces del ecosistema web actual:

* **Meta-framework:** Astro 6.x (Modo SSR habilitado & Vite 6 Environment API).
* **Estilos:** Tailwind CSS v4.3 (Arquitectura *CSS-first* nativa configurada mediante `@theme` en `global.css`).
* **Interactividad:** React 18 + TypeScript (Hidratación selectiva y aislada mediante *Astro Islands*).
* **Base de Datos:** Turso (Edge SQLite) para persistencia perimetral ultra rápida.
* **ORM:** Drizzle ORM (Esquemas y consultas fuertemente tipados).
* **Gestor de Paquetes:** `pnpm` (Estructura de almacenamiento estricta, segura y veloz).

---

## 🛡️ Control de Calidad y Git Hooks (Guardrails)

Para asegurar la estabilidad absoluta del entorno de desarrollo automatizado, el repositorio cuenta con un pipeline estricto de integración local:

* **Husky:** Gestión automatizada de ganchos de Git (`pre-commit`).
* **lint-staged:** Restringe la ejecución de linters únicamente a los archivos modificados en el entorno de pruebas (Staging Area).
* **Formateo y Estilo:** Validación obligatoria mediante **ESLint** y **Prettier** antes de consolidar cualquier avance.
* **Historial Limpio:** Flujo de trabajo basado en **Conventional Commits** (`feat:`, `fix:`, `docs:`, `style:`, `asset:`) para un rastreo atómico y legible del progreso de la aplicación.
